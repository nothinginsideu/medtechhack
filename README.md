# medpartners - нормализация и поиск мед услуг

короче это проект для нормализации прайс-листов клиник. умеет жрать эксельки, пдфки и ворд, вытаскивать оттуда услуги, приводить к единому справочнику через нечеткий поиск и показывать юзеру красивый поиск с фильтрами и сравнением цен.

## стек технологий
- бэкенд: python, fastapi, sqlalchemy, postgresql, rapidfuzz для поиска
- фронтенд: react, vite, tailwind, lucide-react
- докер для деплоя

## как запустить

на компе должен быть установлен докер и докер-компоуз.

1. стяни код с гита
2. в корне проекта лежит docker-compose.yml. просто запусти команду:
```bash
docker-compose up --build -d
```
3. докер сам соберет образы бэка и фронта, поднимет базу данных и настроит порты.
4. бэк будет доступен по адресу http://localhost:8000
5. фронт откроется на http://localhost:5173 (ну или посмотри в консоле какой порт вите выдал)

база данных сама создаст таблицы при первом старте.

## как юзать
- главная страница — это поиск услуг для клиентов. можно вбить например "кровь" или "терапевт"
- там же есть переключатель резидент / нерезидент и инфа о клиниках
- панель оператора (админка) доступна на фронте по вкладке или роуту `/admin`. туда можно залить новые прайсы (пдф, докс, кслс), посмотреть очереди сопоставления и подтвердить руками то, что ИИ не смог сопоставить со 100% уверенностью.
