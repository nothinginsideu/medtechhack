# MedPartners

Платформа для сравнения цен на медицинские услуги по клиникам Казахстана.

Проект разработан в рамках хакатона. Основная идея — клиники присылают свои прайс-листы в разных форматах (Excel, PDF, Word), система их парсит, нормализует названия услуг через нечёткое сопоставление с единым справочником и выдаёт клиенту удобный поиск с возможностью сравнить цены между клиниками.

## Стек

**Бэкенд:** Python 3.12, FastAPI, SQLAlchemy (async), PostgreSQL, RapidFuzz  
**Фронтенд:** React, Vite, Tailwind CSS, Lucide React  
**Инфраструктура:** Docker, Docker Compose  
**ИИ:** Google Gemini API (ассистент в чате)

## Как запустить

Нужен установленый Docker и Docker Compose.

```bash
git clone https://github.com/nothinginsideu/medtechhack.git
cd medtechhack
docker-compose up --build -d
```

После запуска:
- Фронтенд: http://localhost:5173
- Бэкенд (API): http://localhost:8000
- Документация API: http://localhost:8000/docs

База данных создаётся автоматически при первом запуске, справочник услуг подтягивается из Excel файла в `backend/data/`.

## Структура проекта

```
backend/          — FastAPI приложение, парсеры прайсов, алгоритм сопоставления
frontend/         — React SPA, клиентский поиск и панель оператора
data/             — справочник услуг
прайсы/           — тестовые прайс-листы клиник
docker-compose.yml
```

## Основной функционал

**Для клиента** — поиск услуг по названию, сравнение цен между клиниками, фильтр по резидент/нерезидент, график динамики цен, чат с ИИ-ассистентом.

**Для оператора** — загрузка прайс-листов (поддержка xlsx, xls, pdf, docx, zip-архивов), панель верификации сопоставленных услуг, экспорт отчётов в CSV, статистика по загрузкам.
